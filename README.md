# Missing Values Handling in Data Science

## 📌 Project Overview
This project demonstrates how to handle missing values in a dataset using different techniques such as imputation and deletion.

---

## ⚙️ Steps
- Load dataset
- Detect missing values
- Visualize data distribution
- Handle missing values using:
  - Median imputation
  - Dropping missing rows

---

## 🧠 Concepts Covered
- Data Cleaning
- Missing Value Imputation
- Dropna method
- Data Visualization

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 🚀 Goal
To understand different strategies for handling missing data in real-world datasets.
